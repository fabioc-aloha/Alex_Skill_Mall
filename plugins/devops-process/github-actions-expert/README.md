# GitHub Actions Expert

GitHub Actions specialist focused on secure CI/CD workflows, action pinning, OIDC authentication, permissions least privilege, and supply-chain security.

## Installation

Copy `AGENT.md` to `.github/agents/github-actions-expert.agent.md` in your project.

## What's Included

### Agent

| Agent | Description |
|-------|-------------|
| `github-actions-expert` | Security-first CI/CD workflow design with SHA pinning, OIDC, and least-privilege permissions |
