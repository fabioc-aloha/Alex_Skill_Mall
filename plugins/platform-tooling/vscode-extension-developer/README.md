# vscode-extension-developer

Agent for developing VS Code extensions with LSP integration, webview panels, activation optimization, bundling, testing, and Marketplace publishing.

## Install

Copy `AGENT.md` to `.github/agents/vscode-extension-developer.agent.md` in your workspace.

## Usage

Invoke via `@vscode-extension-developer` in chat or as a subagent when building or maintaining VS Code extensions.
