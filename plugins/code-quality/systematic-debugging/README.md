# systematic-debugging

Structured 4-phase debugging agent: problem assessment, root cause investigation, targeted fix, quality assurance with regression tests.

## Install

Copy `AGENT.md` to `.github/agents/systematic-debugger.agent.md` in your workspace.

## Usage

Invoke via `@systematic-debugger` in chat or as a subagent when diagnosing non-trivial bugs.

## Source

Adapted from `awesome-copilot/agents/debug.agent.md` and `awesome-claude-code-toolkit/plugins/bug-detective/`.
