# Context Architect

Plans and executes multi-file changes by mapping context, tracing dependencies, identifying patterns, and sequencing modifications before touching code.

## Installation

Copy `AGENT.md` to `.github/agents/context-architect.agent.md` in your project.

## What's Included

### Agent

| Agent | Description |
|-------|-------------|
| `context-architect` | Maps dependencies, traces ripple effects, and plans multi-file change sequences before modifying code |
